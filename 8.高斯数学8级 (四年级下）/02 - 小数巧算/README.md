# 🧮 小数巧算 (Smart Decimal Calculation)

本项目是 `Gauss` 数学算法库的一个子模块，专注于利用数学定律（交换律、结合律、分配律）实现小数的简便运算。通过 Python 代码模拟人类的“凑整”思维，提升计算效率与准确度。

---

## 🌟 核心理念：AI + 简便算法

在本项目中，我们利用 **AI** 辅助提取算式中的“特征数”，模仿高斯对数字的直觉，将复杂的计算过程转化为“凑整”逻辑。

### 巧算场景：

* **凑整法**：例如  或 。
* **分配律逆用**：例如 。
* **基准数法**：处理一组接近某个整数的小数加减法。

---

## 🤖 AI 学习与实现路径

1. **算式解析**：利用 AI 识别字符串中的特定数字组合（如 `25`, `125`, `4`, `8` 的倍数）。
2. **逻辑生成**：通过 Prompt 引导 AI 生成符合“简便运算”步骤的代码，而非直接进行浮点数运算。
3. **精度校准**：针对 Python 浮点数精度问题（如 `0.1 + 0.2 != 0.3`），AI 辅助引入 `decimal` 模块进行高精度处理。

> **AI Prompt 示例**：
> *"请写一个 Python 函数，能够识别形如 'a * b + a * c' 的字符串算式，并应用乘法分配律将其转化为 'a * (b + c)' 的逻辑进行计算，输出详细的巧算步骤。"*

---

## 🛠️ 代码实现示例

```python
from decimal import Decimal

def smart_distributive_law(a, b, c):
    """
    模拟乘法分配律逆用：a*b + a*c = a*(b+c)
    """
    # 使用 Decimal 避免浮点数精度陷阱
    a, b, c = Decimal(str(a)), Decimal(str(b)), Decimal(str(c))
    
    print(f"检测到公因数: {a}")
    print(f"巧算过程: {a} * ({b} + {c})")
    
    result = a * (b + c)
    return float(result)

# 示例：3.14 * 7.2 + 3.14 * 2.8
print(f"结果: {smart_distributive_law(3.14, 7.2, 2.8)}")

```
